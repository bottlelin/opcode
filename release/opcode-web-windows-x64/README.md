# Opcode - Web Server (Windows x64)

## About
Opcode Web Server provides a web-based interface for Claude Code, allowing you to access Opcode from any device on your network.

## Installation
1. Download and extract the ZIP file
2. Run `opcode-web.exe` from command line or double-click
3. Access the web interface at http://localhost:8080

## Usage
```bash
# Run on default port 8080
opcode-web.exe

# Run on custom port
opcode-web.exe --port 3000

# Show help
opcode-web.exe --help
```

## Requirements
- Windows 10/11 (x64)
- Network access for remote connections

## Features
- Web-based Claude Code interface
- Access from any device on your network
- REST API for programmatic access
- WebSocket support for real-time updates
